# eng-blogs

Evening engineering-blog digest to Telegram (~19:07 IST daily, backup
20:07, via GitHub Actions). 15 company engineering blogs — data platforms,
systems at scale, product & ML engineering — filtered and summarized for
a data engineer. Silent on days with no new posts.

Part of the personal-agents fleet: one agent, one task, one bot.

Secrets (Settings → Secrets → Actions): `ANTHROPIC_API_KEY`,
`TELEGRAM_BOT_TOKEN`, `TELEGRAM_CHAT_ID`.
